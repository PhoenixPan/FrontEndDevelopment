##Attributes: http://www.w3school.com.cn/tags/html_ref_standardattributes.asp
1. Alignment: align="center"  
2. Background: bgcolor="yellow"  
3. Quotation: use either " or '. Use ' when " is in text: name='Bill "HelloWorld" Gates'  
4. Divide line: 
```
<p>Paragraph A</p>
<hr />
<p>Paragraph B</p>
```
Depreciated:
```
align
bgcolor
color
```
